# Clube de Fórmulas — Prof. Samir Tannuri

Marketing and portfolio site for Prof. Samir Tannuri — CNPq researcher, industrial consultant, and speaker specializing in cosmetic R&D.

## Technologies

- Pure HTML/CSS/JavaScript (standalone, no build step)
- Google Fonts (Instrument Serif + Outfit)
- Deployed via Netlify (static hosting)

## Running Locally

Simply open `public/index.html` in a browser, or serve the `public/` directory with any static server:

```bash
npx serve public
```

## Deployment

The site is deployed on Netlify directly from the `public/` directory with no build command required.
